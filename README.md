# 📰 Tech Digest KR

한국어 기술 블로그(velog, tistory) RSS를 수집하고, LLM으로 3줄 요약을 생성해 관심 태그별 개인 뉴스레터처럼 보여주는 서비스입니다.

## ✨ 주요 기능

- 📡 **RSS 수집**: velog, tistory 등 한국어 기술 블로그 자동 수집
- 🤖 **LLM 요약**: 각 글을 3줄로 요약
- 🏷️ **태그 필터링**: 관심 태그별 개인화 피드 구성
- 🧠 **임베딩 분류**: 읽은 글과 유사한 글 vs 새로운 분야의 글 구분

## 🛠️ 기술 스택

- **Backend**: Python, FastAPI
- **LLM**: OpenAI API (교체 가능)
- **임베딩**: sentence-transformers
- **저장**: SQLite
- **스케줄링**: APScheduler
